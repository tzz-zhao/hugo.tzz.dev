Mac で Command Not Found Code 問題の解決方法

  1. Home
  2. ツール
この記事を読むのに必要な時間は約1分です。

環境

Mac OS

やりたいこと

code . コマンドで現在のフォルダをVScodeで開く
code test コマンドで test フォルダをVScodeで開く

問題

command not found: code エラーになる

原因

code コマンドインストールしてない

解決方法

  1. VScode を起動
  2. コマンドパレットを開く（F1）
  3. install code を入力
  4. シェルコマンド：PATH 内に 'code' コマンドをインストールします が表示されるはずなので、選択してインストールします。

以上で完了、無事 code コマンドが使えるようになるはず。
